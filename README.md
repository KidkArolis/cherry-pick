# cherry-pick

A mini GitHub clone demonstrating how to use [Cherrytree](https://github.com/QubitProducts/cherrytree) router with React.js.

It's work in progress, this is an initial version.

# Run locally

```
npm install
npm start
open http://localhost:8080/bundle
```
